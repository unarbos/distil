# Archived v1 dashboard components

These were the components that powered the dashboard before the
v2 monochrome redo. None of them are imported by `app/` after the
v2 cutover (commit `05b7105`); we keep them in `_archive/` for one
release cycle in case we need to restore something. Safe to delete
in a follow-up sweep.

If you find yourself importing from `_archive/`, you almost
certainly want the equivalent component in `components/v2/` instead.
